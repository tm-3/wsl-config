### Proxy

* APT - 
  * create a file /etc/apt/apt.conf and put this in:
  * Acquire::http::Proxy "proxy"

* Environment - 
  * .bashrc or .profile
  * export http_proxy ....

* Git 
  * git config --global http.proxy proxy
